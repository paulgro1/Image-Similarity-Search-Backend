You need to install MongoDB!
Create file in root directory named ".env"
set variables with syntax: <name>=<value>

The following lines contain variables with example values, which can be copied into the .env file.

# Server
BACKEND_HOST=0.0.0.0
BACKEND_PORT=8080

# Database
DATABASE_CLIENT=mongodb://localhost:27017/
DATABASE_NAME=ISS_BACKEND_DATABASE

# Dataset
UPLOAD_FOLDER=./upload
# Main directory
DATA_FOLDER=./data
# Full images folder
DATA_FULLSIZE_FOLDER=fullsize
# Thumbnails folder
DATA_THUMBNAILS_FOLDER=thumbnails
