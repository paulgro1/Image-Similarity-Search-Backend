Installation:
Open Anaconda Prompt
Navigate to target directory with environment.yml file in it
enter: 
    "conda env create -f environment.yml" 
    "conda activate iss-backend"
    "code"
in VSCode > api > app.py:
    View > Command Palette > "Python: Select Interpreter" > "~/anaconda3/envs/issbackend/python.exe"
    (In Windows: Windows + Shift + P)

Alternative:
    "conda create --name iss-backend"
    "conda activate iss-backend"
    "conda install -c pytorch faiss-cpu"
    "conda install -c anaconda flask"
    "conda install -c anaconda flask-cors"
    "conda install -c anaconda sqlalchemy"
    "conda install -c conda-forge flask-restful"

Export environment (current active environment):
    "conda env export --from-history > environment.yml"

Remove environment (if currently active -> conda deactivate):
    "conda env remove --name <name>"

Deactivate environment:
    "conda deactivate"

List all environments:
    "conda env list"

Start VSCode:
    ("conda activate iss-backend")
    "code"
    Check Interpreter!
